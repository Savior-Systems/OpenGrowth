# OpenGrowth Launch Assets

This directory contains drafts and planning materials for the public launch of OpenGrowth.

## Contents

- **[demo-script.md](./demo-script.md)** — A 90-second voiceover script for a demo video showing CLI, dashboard, actions, and strategies.
- **[screenshot-checklist.md](./screenshot-checklist.md)** — Checklist for the key visual screenshots needed for the launch.
- **[gif-storyboard.md](./gif-storyboard.md)** — Storyboard sequence for a 20-second animated GIF showing OpenGrowth in action.
- **[product-hunt.md](./product-hunt.md)** — Product Hunt description, tagline, makers comments, and first-comment drafts.
- **[hacker-news-show-hn.md](./hacker-news-show-hn.md)** — Grounded, honest Show HN post focusing on technical implementations, rules, and limitations.
- **[reddit-posts.md](./reddit-posts.md)** — Customized, developer-friendly reddit drafts for subreddits like `r/opensource`, `r/webdev`, `r/SideProject`, `r/marketing`, and `r/SaaS`.
- **[linkedin-post.md](./linkedin-post.md)** — Professional narrative detailing why OpenGrowth was built and how it can help founders/agencies.
- **[x-thread.md](./x-thread.md)** — A 7-10 post Twitter/X thread introducing the product features.
- **[community-outreach.md](./community-outreach.md)** — Directory of ethical places to share OpenGrowth and the rules of engagement.
